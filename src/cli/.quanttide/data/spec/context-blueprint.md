# context

## 处理步骤

| 步骤名 | 输入 | 输出 | 处理逻辑 | 依赖 |
|--------|------|------|----------|------|
| 步骤名 | 输入(from) | 输出(to) | 处理逻辑(desc) | 依赖(depends) |
| PDF解析与文本提取 | 原始PDF文件 | 页面文本及表格区域 | 使用OCR或PDF解析库读取各页文本，标记每段文字所在的页码和坐标区域 | - |
| 指标识别与映射 | 页面文本及表格区域 | 指标映射结果 | 根据预设的机构-标准指标词典，识别原始指标名称并统一映射为标准指标名（如“角质层水分含量”→“皮肤水分含量”） | PDF解析与文本提取 |
| 数据抽取与结构化 | 指标映射结果 | 结构化观测记录 | 从文本中提取每个样品、时间点、组别的检测值（均值、标准差、变化率、P值），关联原始页码和区域信息 | 指标识别与映射 |
| 数值精度与追溯校验 | 结构化观测记录 | 校验后的观测记录 | 保留原始数值精度（如30.37%不变），标记缺失或异常数值，确保每个值能追溯到原始PDF的页码位置 | 数据抽取与结构化 |
| 长表生成与输出 | 校验后的观测记录 | 最终CSV文件 | 按“报告ID、样品、标准指标名、时间点、组别、均值、标准差、变化率、P值、原始页码、原始区域”长表格式输出CSV | 数值精度与追溯校验 |
