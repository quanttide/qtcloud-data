# 产品路线图

## v0.1.0：规范探索与首个生成单元验证

### 目标

建立可驱动 AI 自动生成处理器组件的规范（Spec）形式，并通过一个具体生成单元验证其可行性。

### 范围

- 聚焦单一生成任务（例如：从指令集描述生成解码逻辑 RTL）
- 定义该任务的最小输入规范（Spec 示例）
- 实现一个脚本/工具，接收 Spec → 输出代码
- 配套：Handbook（含输入格式、假设、失败模式）、Use Case（含蓝图片段与生成结果）

### 成功标准

- 给定符合规范的输入，AI 能稳定生成可编译/仿真的代码；
- 若失败，能定位是 Spec 不足还是生成能力不足，并迭代 Spec。
