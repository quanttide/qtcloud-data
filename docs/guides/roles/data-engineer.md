# 数据工程师指南

数据工程师是负责设计和实施数据架构、数据流水线和数据处理系统的专业人士。

## 核心技能

### 技术技能
- **数据建模**：设计数据模型，理解ER模型、维度建模
- **ETL工具**：Informatica、Talend、Apache Airflow
- **数据库管理**：关系型数据库（Oracle、MySQL、PostgreSQL）、NoSQL（MongoDB、Redis）
- **编程语言**：Python、Java、Scala、SQL
- **大数据技术**：Hadoop、Spark、Kafka

### 专业技能
- **数据清洗**：处理缺失值、异常值、重复数据
- **数据转换**：格式转换、数据聚合、数据关联
- **性能优化**：查询优化、索引设计、分区策略
- **数据安全**：数据加密、访问控制、审计日志

## 工作内容

- 设计和构建数据架构
- 构建数据流水线
- 开发和维护数据处理系统
- 监控和优化数据流程

## 学习路径

1. 掌握SQL和至少一种编程语言（推荐Python）
2. 理解关系型数据库原理和优化
3. 学习ETL概念和工具
4. 掌握大数据处理框架（如Spark）
5. 学习云平台服务（如AWS、Azure、GCP）
