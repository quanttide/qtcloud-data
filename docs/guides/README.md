# 学习指南

本目录提供数据工程领域的基础知识和学习路径，帮助用户理解核心概念和技能。

## 目录

### 概念基础（concepts/）
- [数据工程](concepts/data-engineering.md) - 数据工程领域的概述
- [ETL](concepts/etl.md) - 数据提取、转换和加载流程
- [数据编排](concepts/data-orchestration.md) - 数据任务调度和管理

### 角色指南（roles/）
- [数据工程师](roles/data-engineer.md) - 数据工程师的技能和学习路径
- [数据分析师](roles/data-analyst.md) - 数据分析师的技能和学习路径

## 使用建议

如果你是...
- **新手**：从概念基础开始，建立整体认知
- **数据工程师**：重点关注技术技能和工具链
- **数据分析师**：重点关注分析方法和可视化
- **管理者**：了解核心概念以便团队协作
