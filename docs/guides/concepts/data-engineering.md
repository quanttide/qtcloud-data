# 数据工程

数据工程是构建、维护和管理数据基础设施的领域，包括支持数据的收集、存储、处理、转换和传输，以满足数据分析、业务洞察和决策支持等需求。

## 核心要素

- **数据收集**：从各种数据源获取原始数据
- **数据存储**：选择合适的数据存储方案（数据库、数据仓库、数据湖）
- **数据处理**：通过ETL/ELT流程清洗、转换数据
- **数据传输**：确保数据在不同系统间高效流动
- **数据质量**：监控和维护数据准确性、完整性、一致性

## 常见架构模式

- **批量处理架构**：定时任务处理历史数据
- **流处理架构**：实时处理持续产生的数据
- **Lambda架构**：结合批处理和流处理
- **Kappa架构**：纯流处理架构

## 与数据科学的关系

数据工程为数据科学提供基础设施和高质量数据，数据科学在此基础上进行分析、建模和预测。
