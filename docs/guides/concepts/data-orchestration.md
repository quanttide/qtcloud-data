# 数据编排

数据编排是指协调和管理多个数据处理任务的执行顺序、依赖关系和调度策略，确保数据流按照预定规则高效运行。

## 核心能力

**依赖管理**
- 定义任务间的依赖关系（A→B→C）
- 处理串行和并行执行
- 支持条件分支和循环

**调度策略**
- 定时调度（cron表达式）
- 事件触发（文件到达、消息到达）
- 手动触发

**可观测性**
- 监控任务执行状态
- 记录运行日志
- 收集性能指标

**可控制性**
- 手动干预和重试
- 任务暂停和恢复
- 参数动态调整

## 常见工具

- Airflow：工作流调度平台
- dbt：数据转换框架
- Prefect：现代工作流编排
- Dagster：数据编排工具

## 应用场景

- ETL/ELT流水线
- 机器学习模型训练流水线
- 数据质量检查流水线
- 报表生成自动化
